Three Layers
3 vantages

Each book covers a different layer. They share one stance: don't judge from outside, observe what's already there.

3冊はそれぞれ異なる層を扱う。立場は一つ —— 外から裁かず、そこにあるものを観測する。

L1 / Trace
Git Archaeology
Git考古学

Git log and blame are the archaeological record of what actually survived. We observe impact from traces, not from self-reports.

git log と git blame は、実際に残ったものの考古学的記録。自己申告ではなく痕跡から、インパクトを観測する。

L2 / Structure
Structure-Driven Org
構造駆動エンジニアリング組織論

Organizations break at the same places repeatedly because the subject is always "people." We switch the subject to structure.

組織は同じ場所で壊れ続ける。主語がずっと「人」のままだからだ。主語を構造に切り替える。

L3 / Interior
Psychological OS
心理OS

Structure keeps organizations from breaking people. Psychological OS keeps success and consensus from breaking you.

組織が人を壊さないために構造駆動が要る。成功や正論が自分を壊さないために心理OSが要る。
